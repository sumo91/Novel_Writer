# Chapter 2 Brief: 一包盐换一块灵石

## Chapter Goal

兑现第 1 章章末钩子：陈安完成第一笔两界交易，确认现代细盐和打火机在青石坊有价值；同时引出青石坊巡卫对“无灵力法器”的怀疑。

## V3.2 Outline Reference

- Reference chain: master -> volume -> arc -> unit -> chapter brief.
- Outline obligations: serve volume_001 / arc_001 / unit_0001 by proving the first trade, setting the first goods-price anchor, and introducing market-rule pressure.
- Retrospective note: this brief was written before V3.2 and is annotated here without changing accepted chapter content.

## Opening Hook

灰袍散修许青拿着一枚碎灵，急着要买细盐和火折。陈安不知道碎灵价值，只能靠爷爷旧账本和现实采购常识临场定价。

## Required Beats

1. 陈安先不跨出门，只站在门内交易，确认后门门槛是安全边界。
2. 许青解释细盐能给受伤同伴配药，也能在坊市卖出差价。
3. 陈安用一包加碘盐、两只一次性打火机换到一枚碎灵。
4. 打火机第一次点燃时，许青误以为是无灵力火器，震惊但不敢声张。
5. 陈安追加交易：用一支手电筒试探价格，换到第二枚碎灵和一束黄芽草。
6. 远处青石坊巡卫注意到不需灵力也能发光的手电筒。
7. 子时一个时辰将尽，后门开始变冷，陈安被迫结束交易。

## Character Movement

- 陈安从震惊转为快速算账，第一次显出经营型主角的主动性。
- 许青不是无脑工具人，他识货、急需货，但也害怕坊市规矩。
- 巡卫只露出风险，不在本章正面抓人。

## Ending Pull

许青刚走，巡卫在巷口拦住他，问：“你手里那件无灵力法器，从哪来的？”陈安在门内听见，后门同时合上。

## Continuity Notes

- 后门每天午夜开启一个时辰。
- 第一交易锚点：细盐、打火机、手电筒。
- 碎灵暂定为低阶灵石碎片，一枚碎灵不是大财富，但足以证明交易价值。
- 黄芽草留作第 3 章现实检测或药材价值入口。
