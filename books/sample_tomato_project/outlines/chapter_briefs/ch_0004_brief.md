# 第 4 章 Brief：十分钟会面

## 1. Chapter goal

何盛在发布会暂停后的十分钟窗口内约见林照，试图用“保全公司、保全融资、给林照留位置”的温和交易把公开打脸压回资本可控范围；林照拒绝签临时封口/澄清口径，并用逆光提示发现何盛话术里存在与父亲旧案相似的合同漏洞。

## 2. Required beats

- 开场承接第 3 章：发布会暂停，后台混乱，秦暮雨要求林照解释他为何总能找到验证路径。
- 何盛的人请林照和秦暮雨进入一间临时贵宾室；周衍不在场，但他的压力通过电话和工作人员传来。
- 何盛保持温和，不威胁，只谈“成年人解决问题”：恢复林照顾问身份、给补偿、要求他配合说成后台口径误会。
- 逆光系统只标出商业话术风险和合同漏洞，不直接揭示何盛秘密。
- 林照发现何盛给出的临时协议里有“追溯确认全部试点数据”和“放弃后续异议”的条款，与父亲当年背锅逻辑相似。
- 秦暮雨注意到林照不是情绪反抗，而是在逐条核验条款；她对他的怀疑从“你有秘密”转为“你可能真的能抓住谎言结构”。
- 林照拒绝交易，但不当场撕破脸；他要求保全日志并让东澜医药客户代表出具真实验收状态。
- 何盛第一次明确表示：如果林照继续推进，伤到的不只是周衍，而是很多人的账。
- 章末钩子：秦暮雨把一份三年前旧案合同摘要发给林照，里面出现与何盛临时协议相同的“追溯确认”条款。

## 3. Required emotional payoff

读者要看到林照没有被资本补偿和体面位置收买，且他不是靠莽撞硬刚，而是抓住合同条款里的背锅结构，第一次把父亲旧案和眼前融资局形成相似性连接。

## 4. Required canon references

- 第 3 章发布会已暂停五分钟，何盛要求会后十分钟内见林照。
- `obsidian_capital_he01` 仍然只能作为曜石资本相关复核账号线索。
- 逆光系统只能提示商业谎言、合同漏洞、数据造假和利益关系的可验证矛盾。
- “信任资源消耗”仍是待批准的系统代价方向，不能写成完整规则。

## 5. Open threads touched

- `thread_001`: 周衍背锅结构被推进到临时协议层面。
- `thread_002`: 系统开始在合同条款中提示漏洞，并加重秦暮雨对林照秘密的追问。
- `thread_003`: 何盛亲自出面，旧案合同摘要与当前条款产生相似性，但仍不证明主谋身份。

## 6. End hook

秦暮雨发来的旧案合同摘要里，出现和何盛临时协议几乎相同的“追溯确认”条款；林照意识到父亲当年的坑，可能不是第一次出现。
