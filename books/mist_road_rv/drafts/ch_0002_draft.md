# Chapter 2 Draft: 第一条上车规则

## 1. Chapter Draft

房车发动机一响，车外的人群也跟着炸了。

“他车能动！”

“开门！带我们走！”

“我爸还在外面，你不能自己跑！”

一只手拍在驾驶座窗外，掌心带血，拖出一条红印。另一边，有人绕到车尾，蹲下去摸油箱盖。

陆沉舟没有立刻踩油门。

中控屏上的倒计时还在跳。

【车内安全空气：18分07秒。】

【任务一：清理 C 区加油口。】

【目标：取得净雾滤芯组件。】

C 区在服务区最靠外的一排加油口。

离雾墙最近。

也离那几具重新站起来的东西最近。

陆沉舟从后视镜里看见油箱旁边那个人已经撬开外盖，正把一根塑料管往里塞。

他按下车窗。

“手拿出来。”

那人抬头，脸上全是汗：“你那么大一辆车，一箱油够你跑几十公里，分点怎么了？”

陆沉舟看了他一眼。

“我说手拿出来。”

“不给油你也别想走！”那人咬牙，“大家都困在这儿，你凭什么一个人开车跑？”

陆沉舟拉开车门，下车。

他动作不快。

可他一下车，堵在门口的人群反而退了半步。

刚才那几个被他从车祸里拖出来的人都看着他。有人感激，有人害怕，有人眼里已经开始算计房车还有多少位置。

陆沉舟走到车尾，扣住偷油那人的手腕，把塑料管从油箱口拔出来。

汽油味一下冲出来。

“现在雾里有明火，有撞毁车辆，有不知道是什么东西的感染者。”陆沉舟把塑料管扔到地上，“你在这里抽油，是想把所有人一起点了？”

那人脸色涨红：“我、我也是为了活！”

“想活就离油箱远点。”

陆沉舟松手。

那人踉跄后退，撞到旁边一个抱孩子的女人。

女人立刻哭喊起来：“你别吓他！他只是想找点油！我孩子受伤了，你车里有地方，你先让孩子上车！”

她把孩子往前推。

孩子额头破了，哭声已经哑了。

人群像找到了突破口，七嘴八舌压上来。

“孩子总能上吧？”

“老人也得上！”

“你刚才不是救人吗？救一半算什么？”

陆沉舟的视线从孩子脸上扫过，又落到女人身后。

那里有个中年男人扶着胳膊，袖子被血浸透，眼睛却一直盯着房车驾驶室。旁边两个年轻人互相递了个眼神，正往副驾驶那侧挪。

“退后。”

没人退。

倒计时还在跳。

17分39秒。

陆沉舟抬手，指向服务区入口。

“受伤的去入口右侧。能走的自己走，不能走的找人抬。别堵车门。”

“你凭什么安排？”

“凭我车能动。”

这句话一落，人群更躁。

就在这时，服务区入口传来一声短促的呵斥。

“别围着车！出血的先压迫止血，喘不上气的侧卧，发灰的别往人堆里抬！”

那声音陆沉舟听过。

事故刚发生时，就是这道声音在喊“别拔玻璃”。

他转头。

服务区入口的水泥墩旁，一个女人跪在地上，膝边摊着急救包。她穿一件灰色冲锋衣，袖口卷到手肘，手上套着一只一次性手套，另一只手套已经破了，指尖沾着血。

她用止血带勒住伤员大腿，抬头扫了一眼人群。

“那个抱孩子的，孩子还能哭，先按住额头，不要往车前挤。你后面那个男的，把袖子掀开。”

中年男人脸色一变。

“我没事。”

女人声音冷下来：“袖子掀开。”

中年男人没动。

陆沉舟看向他。

那人手臂上的血不是往下流，是沿着伤口边缘一圈圈洇开。灰白色的絮状物沾在皮肉上，像潮湿的棉线。

女人站起身。

“他不能和普通外伤待一起。”

人群瞬间退开。

“什么意思？”

“他也会变成刚才那种东西？”

“你别胡说！”

中年男人急了，抬手就要抓女人的急救包：“你少吓人，我只是被玻璃划了一下！”

陆沉舟一步过去，挡在女人和那人之间。

“叫什么？”

女人没有立刻回答。

她先看了一眼他手里的破窗锤，又看了一眼房车，再看向他眼睛。

“林知夏。急诊。”

陆沉舟点头。

“陆沉舟。”

“我知道。”林知夏说，“刚才那边有人喊你陆队。”

她说完，指向中年男人的胳膊。

“这个人不能上你的车，也不能进大厅。他伤口边缘发灰，瞳孔反光不对，刚才还在避开灯。”

中年男人怒了：“你凭什么说我有问题？你有仪器吗？你是医生还是神仙？”

林知夏把剩下那只破手套摘下来，扔到一边。

“我是急诊护士。不是神仙，所以我只说风险。”

人群里有人小声说：“护士又不是医生……”

林知夏看过去。

“那你来判断谁会咬人？”

那人闭嘴。

陆沉舟看了她一眼。

这句话稳。

不解释太多，不承诺全救，只说风险。

和他一样，是现场里还知道自己能做什么的人。

车内中控忽然发出一声轻响。

【车内安全空气：16分58秒。】

【任务目标距离：C 区加油口，47米。】

【污染浓度上升。】

陆沉舟关上车门。

林知夏听见声音，视线落到中控屏上。

她没有问“那是什么”。

她只问：“你要去 C 区？”

陆沉舟看向加油口。

灰白雾正从 C 区的顶棚下往外淌，像一层贴着地面的水。加油机旁边撞着一辆红色轿车，前盖翘起，油枪还插在油箱口，软管绷得很紧。

油枪下方，半个人趴在地上。

看不出死活。

“要拿一个东西。”陆沉舟说。

“什么东西？”

“净雾滤芯组件。”

林知夏皱眉。

她显然没听懂，但没有追问系统。

“你一个人过去，会把雾里那几具都引过来。”

“所以你留这儿，把能动的人往入口右侧分开。”

“我为什么听你的？”

陆沉舟看向她：“因为你刚才也在分诊。”

林知夏顿了一下。

陆沉舟继续：“普通外伤、不能移动的、疑似污染的，分开。别让他们往车边挤。有人抢车，喊我。”

“喊你有用？”

“比喊天有用。”

林知夏看了他半秒。

“我需要胶带、干净布、酒精，最好还有口罩。”

陆沉舟拉开房车侧门，从柜子里扔出一个旧工具包和半包医用口罩。

“口罩只有这个。”

“够压一会儿。”

她接住东西，动作干脆。

抱孩子的女人又哭起来：“你们说完没有？我孩子怎么办？”

林知夏蹲下看了一眼孩子的伤口。

“头皮裂伤，能哭，暂时不是最危险的。你按住这里，别再把他往人堆里推。”

女人崩溃：“你什么意思？我孩子不重要？”

林知夏抬头，眼神很冷静。

“重要。但现在最先死的不是他。”

女人被噎住。

陆沉舟转身往 C 区走。

他没有跑。

地面有油，雾里有东西，跑只会摔。

他把破窗锤插回腰侧，左手拿灭火器，右手拽起路边一根断掉的警示杆。经过红色轿车时，他先踢了一脚地上的空饮料瓶。

瓶子滚进雾里。

咔。

一只手从红色轿车底下伸出来，抓住瓶子。

陆沉舟停住。

那半个人动了。

它上半身从车底慢慢拖出来，后腰以下被压烂，脸却抬着，嘴里发出漏风一样的声音。

服务区入口有人倒吸一口冷气。

陆沉舟没有回头。

“别叫。”

他把灭火器喷口对准那东西的脸。

白雾炸开。

那东西动作一顿，陆沉舟用警示杆卡住它肩膀，把它顶回车底。

同一时间，红色轿车另一侧传来脚步声。

第二个。

陆沉舟拔出油枪。

软管回弹，啪地抽在加油机外壳上。

火花没有起。

他松了一口气，立刻弯腰去看加油机下方的检修盖。

系统没有地图。

只有中控里那句“C 区加油口”。

滤芯组件不可能摆在台面上。

他扫过加油机、油管、旁边的应急箱。

应急箱锁扣已经锈住。

陆沉舟抡起警示杆，砸开锁。

里面没有滤芯，只有吸油棉、旧扳手、密封胶和一只灰色金属盒。

金属盒上贴着一张褪色标签。

【雾化应急过滤件】

陆沉舟伸手去拿。

背后风声一紧。

他侧身。

一只手擦着他的肩膀抓空，指甲在冲锋衣上划出四道白痕。

第二具雾尸扑上来，半张脸被玻璃划开，嘴角却咧得很大。

陆沉舟用肩膀撞上加油机侧边，借力转身，警示杆横扫它膝盖。

雾尸跪下。

没倒。

陆沉舟抓起密封胶罐砸过去，罐体在它脸上爆开，黑灰色胶液糊住眼睛。

它嘶哑地扑腾。

服务区入口那边忽然乱了。

“他过来了！”

“别让他靠近我！”

那个被林知夏指出污染疑似的中年男人想往人群里钻，几个人推搡着躲开。他急了，反手抓住抱孩子女人的胳膊。

“你们凭什么孤立我？我没变！”

孩子哭声陡然尖起来。

林知夏抄起地上的三角警示牌，横在他和人群之间。

“松手。”

“滚开！”

中年男人挥手去打她。

下一秒，一根警示杆从雾边飞过来，砸在他脚边。

陆沉舟的声音跟着压过来。

“再碰她，我先处理你。”

他一手提着灰色金属盒，一手拎着灭火器，从 C 区雾边走出来。肩膀上有一道被抓出来的白痕，袖口沾着密封胶，脸色冷得像服务区顶棚下的水泥。

人群安静了一瞬。

林知夏看见他手里的金属盒。

“拿到了？”

“拿到了。”

陆沉舟走到房车旁，拉开副驾门，把金属盒放进去。

中控屏亮起。

【净雾滤芯组件：1/2。】

【核心过滤件已获取。】

【缺失：加压密封环。】

【提示：组件可临时接入，稳定启动需完成二次固定。】

陆沉舟盯着屏幕两秒。

还差一半。

但至少不是空手。

林知夏也看见了那几行字。

她的眼神变了一下。

这次她问了。

“你的车，是什么情况？”

“现在不是解释的时候。”

“我也这么觉得。”她说，“所以先说规则。”

陆沉舟看向她。

林知夏指了指人群。

“你再不开口，他们会默认这辆车属于所有人。你不开规则，他们就替你定规则。”

陆沉舟沉默半秒。

然后他关上副驾门，站到房车侧门前。

“听清楚。”

人群里有人骂：“你又想说什么？”

陆沉舟没有提高声音。

但他刚从 C 区走回来，手里还握着沾胶的灭火器，刚才想抢油箱的人不自觉闭了嘴。

“这辆车不是救护车，不是收容所，也不是谁嗓门大谁就能上的公交。”

抱孩子的女人脸色变了。

有人怒道：“你没人性！”

“想骂随便。”陆沉舟说，“想活，听规则。”

雾在服务区边缘翻涌。

倒计时还在车里跳。

他一字一句说下去。

“第一，上车不是被救，是签约。”

“第二，能移动，听指挥，不隐瞒伤情和污染症状，才有资格申请上车。”

“第三，上车后服从撤离安排。抢方向盘、抢物资、隐瞒感染，直接下车。”

人群死寂。

林知夏站在他侧后方，没有反驳。

中年男人忽然笑了一声。

“那孩子呢？孩子也签约？”

他一开口，人群像终于找到刀口。

抱孩子的女人立刻跪下去，把孩子往车门前推。

“你让他签！我替他签！你不救他，他就死在你车前！”

孩子额头的血顺着眉骨往下流，哭声越来越弱。

所有视线都压到陆沉舟身上。

有人低声说：“你总不能连孩子都不管吧？”

林知夏看了孩子一眼，眉头皱起。

她刚要开口，陆沉舟先动了。

他从车里拿出一卷纱布和半瓶水，扔给林知夏。

“处理伤口。”

女人眼睛一亮。

陆沉舟接着说：“人不上车。”

女人脸上的亮色僵住。

“为什么？”

“因为他现在能哭，能呼吸，头皮伤不是优先级最高。”陆沉舟看着她，“因为你刚才把他推到车前，是想用他逼我开门。”

女人张了张嘴。

陆沉舟声音压低。

“你想救他，就按她说的压住伤口，别再把他当钥匙。”

林知夏接住纱布。

这一次，她没有说陆沉舟冷血。

她只是蹲下去，按住孩子额头，对女人说：“手给我。按这里，持续压，不要松。”

女人哭着照做。

人群里的骂声低了下去。

但没有消失。

有人往后退。

也有人盯着房车门，眼神更阴。

陆沉舟看见了。

他把侧门关上。

中控屏在车内幽幽亮着。

【净雾滤芯组件：1/2。】

【车内安全空气：11分26秒。】

雾里的影子又近了一点。

## 2. New Facts Introduced

- 林知夏正式登场，是急诊护士，能进行外伤处理、现场分诊和污染疑似判断。
- 净雾滤芯组件被拆分为至少两个部分，本章取得核心过滤件，仍缺加压密封环。
- 房车系统可以通过中控提示任务距离、污染浓度和组件状态。
- 第一条上车规则公开：上车不是被救，是签约。

## 3. State Changes

- 陆沉舟从系统绑定者进一步成为房车规则制定者。
- 林知夏从远景伏笔变为核心队友候选。
- 净雾滤芯核心过滤件进入房车，但完整滤芯尚未稳定启动。
- 服务区人群对房车规则产生明显反弹。
