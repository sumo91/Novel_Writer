# Common Failure Modes

Use this file when reviewing drafts and outlines.

- Formulaic payoff: the chapter uses a known pattern but lacks specific emotional force.
- Passive protagonist: events happen around the lead while others drive the scene.
- Generic opposition: the antagonist exists only to be wrong or foolish.
- Explanation drag: the chapter explains rules before the reader needs them.
- Canon drift: injuries, resources, secrets, locations, or relationships change without cause.
- Thread sprawl: new promises are opened faster than old ones are advanced or closed.
- False cliffhanger: the ending withholds information without creating meaningful stakes.
- Tone mismatch: jokes, romance, violence, or melodrama break the book's stated promise.
