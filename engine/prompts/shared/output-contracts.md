# Shared Output Contracts

All agents must follow these rules.

## Canon Safety

- Treat book-local canon as truth.
- Do not override canon with shared theory or genre patterns.
- If a needed fact is missing, state an assumption instead of inventing permanent canon.
- List every proposed canon change separately.
- Major canon changes are not authoritative until human approval.

## Evidence

Review agents must provide:

- Issue type.
- Severity: low, medium, high, blocker.
- Evidence from draft, brief, or canon.
- Suggested fix.

Avoid vague approval. If a chapter passes, explain what evidence supports the pass.

## Structured Sections

Every agent response should include:

1. Assumptions.
2. Main output.
3. Canon or state changes proposed.
4. Human approval needed.

When JSON is requested, return valid JSON only.
